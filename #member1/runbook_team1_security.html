Generate ECDSA Key Pair

Run the script below from the project root to generate the ECDSA P-256 key pair. Keys are written to the paths defined in sbom_toolsuite/config.json.

⚠️ Never commit private_key.pem to version control. Add it to .gitignore immediately after generation. 
# Run from project root: c:\Study\Projects\BOM\
python -c "
from cryptography.hazmat.primitives.asymmetric import ec
from cryptography.hazmat.primitives import serialization
import os

os.makedirs('sbom_toolsuite', exist_ok=True)

private_key = ec.generate_private_key(ec.SECP256R1())
public_key  = private_key.public_key()

with open('sbom_toolsuite/private_key.pem', 'wb') as f:
    f.write(private_key.private_bytes(
        encoding=serialization.Encoding.PEM,
        format=serialization.PrivateFormat.PKCS8,
        encryption_algorithm=serialization.NoEncryption()
    ))

with open('sbom_toolsuite/public_key.pem', 'wb') as f:
    f.write(public_key.public_bytes(
        encoding=serialization.Encoding.PEM,
        format=serialization.PublicFormat.SubjectPublicKeyInfo
    ))

print('[OK] Keys written to sbom_toolsuite/')
"
✅ Expected output: [OK] Keys written to sbom_toolsuite/
Two files appear: private_key.pem and public_key.pem.

Restrict Private Key Permissions

Lock down the private key so only the scanning service account can read it.

Windows (PowerShell — run as Administrator)

$path = "c:\Study\Projects\BOM\sbom_toolsuite\private_key.pem"
icacls $path /inheritance:r
icacls $path /grant:r "$($env:USERNAME):(R)"

Linux / macOS

chmod 600 sbom_toolsuite/private_key.pem
ℹ️ The public_key.pem can remain world-readable — it is distributed to clients for signature verification.

Verify Key Pair Integrity

Sign a test message and verify it to confirm the key pair works correctly before use in production.

python -c "
from cryptography.hazmat.primitives.asymmetric import ec
from cryptography.hazmat.primitives import hashes, serialization

with open('sbom_toolsuite/private_key.pem', 'rb') as f:
    priv = serialization.load_pem_private_key(f.read(), password=None)
with open('sbom_toolsuite/public_key.pem', 'rb') as f:
    pub = serialization.load_pem_public_key(f.read())

message   = b'SBOM-key-pair-verification-test'
signature = priv.sign(message, ec.ECDSA(hashes.SHA256()))
pub.verify(signature, message, ec.ECDSA(hashes.SHA256()))

print('[OK] Key pair is valid — ECDSA sign/verify passed.')
"
✅ Expected: [OK] Key pair is valid — ECDSA sign/verify passed.
If an exception is raised, regenerate the keys (Section 2) and repeat.
